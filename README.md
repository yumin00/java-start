# Java Start

이 레포지토리는 [김영한의 자바 입문 강의](https://www.inflearn.com/course/%EA%B9%80%EC%98%81%ED%95%9C%EC%9D%98-%EC%9E%90%EB%B0%94-%EC%9E%85%EB%AC%B8)를 듣고 학습한 내용을 정리한 공간입니다.

## 📚 소개
- **목표**: 자바 언어의 기초를 체계적으로 학습하고, 작성한 코드를 정리하여 학습 내용을 복습합니다.
- **내용**: 자바 기초 문법

## 🗂️ 학습 내용
- Hello World
- 변수
- 연산자
- 조건문
- 반복문
- 스코프, 형변환
- 배열
- 메서드


## 🛠️ 환경
- **JDK**: OpenJDK 23
- **IDE**: IntelliJ IDEA
